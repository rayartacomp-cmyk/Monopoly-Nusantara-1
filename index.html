Aplikasi game multiplayer untuk 3+ pemain dengan sistem imposter. Pemain harus menebak kata rahasia sementara imposter berpura-pura tahu.

  • Minimal 3 pemain - Semakin banyak semakin seru!
  • Random imposter - Imposter dipilih secara acak setiap game
  • 15 kategori - Dengan minimal 30 kata per kategori
  • Swipe to reveal - Swipe ke atas untuk melihat kata/imposter
  • Voting system - Temukan siapa imposter di antara kalian

üöÄ Cara Menjalankan

Ini adalah aplikasi React Native. Untuk menjalankannya:

# Clone repository
git clone <repository-url>
cd GameImpostor

# Install dependencies
npm install

# Jalankan dengan Expo
npx expo start

# Scan QR code dengan Expo Go di smartphone

Atau build untuk production:

# Build untuk Android
npx expo run:android

# Build untuk iOS
npx expo run:ios
